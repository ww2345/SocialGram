# frontend `src` for SocialGram\n\nDrop this `src` folder into your existing Vite React project.\n\nQuick steps:\n\n1. Extract this `src` into your `frontend/` folder (delete existing `src` first).\n2. Make sure dependencies are installed:\n   - axios\n   - react-router-dom\n   - socket.io-client\n\n3. Example install (from frontend folder):\n   npm install axios react-router-dom socket.io-client\n\n4. Run dev:\n   npm run dev\n\nNotes:\n- Styling uses plain CSS in index.css (neon theme) — no Tailwind required.\n- If your backend is on a different host/port, set VITE_API_URL in your environment when running Vite:\n  export VITE_API_URL=http://localhost:4000
